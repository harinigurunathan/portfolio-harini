Detection of Fake Profiles on Social Media Using Blockchain Technology

* The project addresses the critical issue of fake profiles on social media platforms, which are often used for spamming, phishing, and other malicious activities. To combat this, our solution leverages blockchain technology for its immutable and decentralized nature, ensuring that user data remains secure and tamper-proof. By storing user registration data such as username, email, phone number, and profile picture on the blockchain, we can create a transparent and verifiable record of user authenticity. Additionally, social media activity, including posts, likes, comments, and bio information, is recorded to provide a comprehensive view of user behavior.

* To enhance the detection process, we analyze behavioral data using machine learning algorithms. This data includes interaction patterns, posting frequency, friend/connection requests, and message content. By training models on this data, we can identify anomalies and patterns indicative of fake profiles.
